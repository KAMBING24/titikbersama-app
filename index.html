Import Data dari Spreadsheet

Upload File CSV/Excel

Drag & drop file CSV dari spreadsheet Anda atau klik untuk browse

Untuk rekap penjualan (mis. sheet "rekap_titikbersama"), biarkan tidak dicentang agar stok di aplikasi tidak berubah.

Petunjuk Import:

  1. Buka Google Spreadsheet (mis. rekap penjualan dengan kolom: No Faktur, Tanggal, Customer, SKU, Produk, Qty, Harga Jual, Total Jual, Harga Modal, Total Modal, Profit, Metode, Status Bayar)
  2. File → Download → CSV (comma-separated values) untuk sheet yang aktif
  3. Upload file .csv di atas (untuk .xlsx, export dulu ke CSV dari Sheets)
  4. Format rekap di atas diimpor sebagai penjualan dan mengisi/memperbarui master barang (SKU, nama, harga jual)
  5. Format sederhana lain: baris dengan kolom supplier dianggap pembelian; selain itu penjualan
Catatan: Baris dengan No Faktur kosong dilewati. Nomor faktur sama tidak diimpor dua kali.
